# Build the Explore page on WordPress

## 1. Create the page

1. **Pages → Add New**
2. Title: **Explore**
3. **Page → Template** (right sidebar): choose **Explore page**
4. Do **not** use the old “Marketing hero” for Explore — use the Explore patterns below.

## 2. Add patterns (top to bottom)

Open **Patterns → CCG Fusion — Explore** and insert **in this order**:

| Order | Pattern name |
|------|----------------|
| 1 | Explore — breadcrumbs *(replace this block after theme 0.3.1 if breadcrumbs look like a numbered list)* |
| 2 | Explore — hero |
| 3 | Explore — platforms grid |
| 4 | Explore — platform roadmap |
| 5 | Explore — What's happening |
| 6 | Explore — Learn & Get Started |
| 7 | Explore — Getting started band |

## 3. Publish

**Update** the page and view it on the site.

## 4. Set URL (optional)

**Settings → Permalinks** — use `/explore/` slug for this page if it should match the prototype.

## Hero image

The hero uses `assets/images/sections/initiatives-hero-cms-gov.png` inside the theme. If the background is missing after upload, ask your developer to copy that PNG from the FUSION Sphere prototype into the theme folder.
