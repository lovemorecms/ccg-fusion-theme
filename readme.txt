Block theme based on the FUSION Sphere prototype.

== Description ==

Starter block theme for marketing pages. Header and footer are controlled by the theme (not the block editor). Editors use block patterns for page sections.

== Installation ==

1. Zip this entire folder (folder name should be ccg-wp-theme).
2. In WordPress admin: Appearance → Themes → Add New → Upload Theme.
3. Activate "CCG Fusion".
4. Assign the "Marketing landing" template to key pages as needed.

See START-HERE.md for non-technical instructions.
